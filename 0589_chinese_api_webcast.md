#### Chinese Revit 2012 API Webcast My colleague Joe Ye of the Beijing office is presenting a Chinese version of the recent [Revit 2012 API Webcast](http://thebuildingcoder.typepad.com/blog/2011/05/revit-2012-api-webcast.html) on Thursday, June 2nd. As always, to register, you can visit the Autodesk Developer Network (ADN) DevTech API [training schedule site](http://www.adskconsulting.com/adn/cs/api_course_sched.php), filter for 'Revit API', and click on the [register](http://usa.autodesk.com/adsk/servlet/item?id=10417086&siteID=123112&cname=Revit%20API,%20Webcast,%20Jun%2002%202011,%20201129) link. This webcast is free of cost and open to all developers, regardless of ADN membership. Here is the Chinese agenda: - Revit 2012 API SDK- 2011 Revit 2012- Revit 2012 APIs API- Revit Structure API- Break- Revit MEP API- SDK samples The English agenda was given in the [previous post](http://thebuildingcoder.typepad.com/blog/2011/05/revit-2012-api-webcast.html).Project Vasari 2.0 An [updated version 2.0 of Project Vasari](http://labs.blogs.com/its_alive_in_the_lab/2011/05/project-vasari-technology-preview-20-now-available.html) is now available, compatible with Revit 2012. It is of interest to Revit API developers, since it [includes part of the API](http://thebuildingcoder.typepad.com/blog/2010/11/project-vasari-api.html).BIM Family Toolkit for Revit 2012 The BIM Family Toolkit has also been updated and is now [available for Inventor 2012 and Revit 2012](http://labs.blogs.com/its_alive_in_the_lab/2011/05/bim-family-toolkit-now-available-for-inventor-2012-and-revit-2012.html).
