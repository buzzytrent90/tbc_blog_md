### Revit Family API Webcast

To register, you can visit the Autodesk Developer Network (ADN) DevTech API
[training schedule site](http://www.adskconsulting.com/adn/cs/api_course_sched.php),
filter for Revit Family API, and click on the
[register](http://usa.autodesk.com/adsk/servlet/item?id=10417086&siteID=123112&cname=Revit%20Family%20API,%20Webcast,%20Jul%2023%202009,%20200841) link.
