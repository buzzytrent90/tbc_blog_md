### ADN Training Material on GitHub and Portathon Reminder

My colleague
[Gopinath Taget](http://adndevblog.typepad.com/aec/gopinath-.html) and
I completed the migration of the ADN Revit API training material in preparation for the
[Sydney Revit API training](http://thebuildingcoder.typepad.com/blog/2013/07/sydney-revit-api-training-and-vacation.html),
even though the training actually ended up based on completely new spontaneus project ideas from the participants instead, making it more fun, challenging and creative.

Anyway, here are some steps of the migration path and pointers to more detailed information:

- [Migrating the ADN training labs to Revit 2014](http://thebuildingcoder.typepad.com/blog/2013/06/migrating-the-adn-training-labs-to-revit-2014.html)
- [The ADN sample AdnRme for Revit MEP 2014](http://thebuildingcoder.typepad.com/blog/2013/06/the-adn-sample-adnrme-for-revit-mep-2014.html)
- [ADN training material for Revit Structure 2014](http://thebuildingcoder.typepad.com/blog/2013/06/adn-training-material-for-revit-structure-2014.html)
- [Recursively disable architecture mismatch warning](http://thebuildingcoder.typepad.com/blog/2013/07/recursively-disable-architecture-mismatch-warning.html)

Gopi now proceeded with the next step, uploading the standard ADN Revit API Training Material to the
[RevitTrainingMaterial](https://github.com/ADN-DevTech/RevitTrainingMaterial) GitHub
repository and making it available to the general public.

#### Portathon Reminder

Here is a reminder of the upcoming inaugural
[Exchange Apps Portathon](http://thebuildingcoder.typepad.com/blog/2013/07/earn-100-submitting-an-autodesk-exchange-app.html) event,
a 48 hour virtual web based coding festival on September 13-14 at which you can earn $100 submitting an Autodesk Exchange App.

You can submit them before then as well, of course, and still be eligible for the reward.

The Portathon is all about helping you get your apps into the store quickly and easily, including localised apps!
