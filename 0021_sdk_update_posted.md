### SDK Update Posted

The Revit 2009 web update 2 has been posted to the Autodesk product site for the English versions of Revit
[Architecture](http://usa.autodesk.com/adsk/servlet/item?siteID=123112&id=11017599),
[MEP](http://usa.autodesk.com/adsk/servlet/item?siteID=123112&id=11273489) and
[Structure](http://usa.autodesk.com/adsk/servlet/item?siteID=123112&id=9281007).
Unlike previous versions, this update makes use of patching technology and does not require a complete reinstallation. For this reason, the SDK is not included in the update, like it used to be in previous versions. Therefore, the Revit 2009 SDK web update 2 has been posted separately. It is available under the link
[Revit 2009 SDK (Update 2) (zip - 61330Kb)](http://images.autodesk.com/adsk/files/revit_2009_sdk.zip)
in the
[Revit Developer Center](http://usa.autodesk.com/adsk/servlet/index?siteID=123112&id=2484975)
as well as on
[ADN](http://adn.autodesk.com/adn/servlet/index?siteID=4814862&id=5017413&linkID=4901650).
