#### Revit 2012 API Webcast I may be away visiting Turkey and climbing [Mount Ararat](http://thebuildingcoder.typepad.com/blog/2011/05/improved-mep-element-shape-and-mount-ararat.html#2), but I can still pass on the occasional tidbit to you at the same time. The Revit 2012 API webcast is scheduled for Thursday, May 19th. To register, you can visit the Autodesk Developer Network (ADN) DevTech API [training schedule site](http://www.adskconsulting.com/adn/cs/api_course_sched.php), filter for 'Revit API', and click on the [register](http://usa.autodesk.com/adsk/servlet/item?id=10417086&siteID=123112&cname=Revit%20API,%20Webcast,%20May%2019%202011,%20201123) link. This webcast is free of cost and open to all developers, regardless of ADN membership. Here is the rough agenda: - Introduction- Migrating Revit plug-ins to Revit 2012- Revit 2012 New APIs - Geometry API- Construction modeling API- Point Cloud API- Updates in Revit Structure API- Break- Revit 2012 New APIs (contd)- Extensible Storage API- Compound Layer Structure API- Updates in Revit MEP API- New SDK samples- Winding up I am looking forward to virtually meeting you there!
