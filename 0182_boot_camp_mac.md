### Revit Boot Camp on Mac

Following up on the
[practical notes on impractical things](http://thebuildingcoder.typepad.com/blog/2009/07/practical-notes-on-impractical-things.html),
here is one on a practical thing as well:

Autodesk is now officially supporting Apple Boot Camp for users of AutoCAD, 3ds Max, Inventor and last but not least the three flavours of Revit.
Boot Camp is a component of the Mac OSX operating system that enables customers to run Windows on the Mac.
See
[www.autodesk.com/mac](http://www.autodesk.com/mac)
for more general details and this
[FAQ](http://usa.autodesk.com/adsk/servlet/ps/dl/item?siteID=123112&id=13578805&linkID=9243099)
for some frequently asked Revit specific questions and hardware requirements.
